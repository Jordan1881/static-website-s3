About This Project

A CI/CD pipeline from code to cloud

What Is This?

A static website hosted on AWS S3 and deployed automatically via GitHub Actions. Every time code is pushed to the main branch, the workflow fires and syncs the latest files to S3 — no manual uploads needed.

The Deployment Flow

git push → GitHub Actions triggers → AWS credentials loaded → aws s3 sync → Live on S3

Security Approach

AWS credentials are stored as encrypted GitHub repository secrets — they never appear in the source code or workflow logs. The IAM user only has permission to interact with the specific S3 bucket, following the principle of least privilege.

Tech Stack

GitHub — source control and Actions CI/CD runner
AWS S3 — static website hosting with public-read access
AWS IAM — scoped deployment credentials
HTML/CSS — the website itself (no frameworks, no build step)
